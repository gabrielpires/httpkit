# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Instead, report them by emailing the maintainer directly. You can find contact information on the GitHub profile [@gabrielpires](https://github.com/gabrielpires).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 72 hours. If the issue is confirmed, a patch will be released as soon as possible.
